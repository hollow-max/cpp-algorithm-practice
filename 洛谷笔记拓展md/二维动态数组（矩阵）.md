# 二维动态数组（矩阵）

```c++
vector<vector<int>> a(n,vector<int>(m));
//创建了一个二维动态数组即矩阵
```

这行代码可以理解为：

1. **外层 `vector<vector<int>>`**：创建一个vector，其中的每个元素都是 `vector<int>` 类型
2. **`a(n, vector<int>(m))`**：调用vector的构造函数
   - 第一个参数 `n`：创建 n 个元素
   - 第二个参数 `vector<int>(m)`：每个元素的初始值是一个包含 m 个 int 的**vector**

等价写法：

```c++
// 方法1：分步创建
vector<vector<int>> a;
for (int i = 0; i < n; i++) {
    vector<int> row(m);  // 创建一行，有m个元素
    a.push_back(row);    // 将这一行加入矩阵
}

// 方法2：先创建外层，再初始化
vector<vector<int>> a(n);  // 创建n行
for (int i = 0; i < n; i++) {
    a[i].resize(m);        // 每行有m列
}

// 方法3：使用上面的单行写法（最简洁）
vector<vector<int>> a(n, vector<int>(m));
```

可以这样理解：

- `vector<int>(m)` 创建了一个有 m 个整数的一维数组
- `vector<vector<int>> a(n, ...)` 创建了 n 个这样的一维数组
- 最终结果就是一个 n×m 的矩阵

示例：

```c++
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n = 3, m = 4;
    
    // 创建一个 3行4列 的矩阵，所有元素初始化为0
    vector<vector<int>> a(n, vector<int>(m));
    
    // 赋值
    a[0][0] = 1;
    a[1][2] = 5;
    a[2][3] = 9;
    
    // 遍历
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            cout << a[i][j] << " ";
        }
        cout << endl;
    }
    
    return 0;
}
```

输出：

```txt
1 0 0 0
0 0 5 0
0 0 0 9
```

VS静态数组

```c++
// 静态数组（不能动态确定大小）
int a[20][20];  // 必须提前知道最大大小

// 动态vector（可以根据输入动态创建）
vector<vector<int>> a(n, vector<int>(m));  // n和m由输入决定
```

**优点**：

- 大小可以在运行时确定
- 自动管理内存
- 可以使用vector的各种方法（如 `.size()`）

初始值自定义

```c++
// 所有元素初始化为0
vector<vector<int>> a(n, vector<int>(m));

// 所有元素初始化为5
vector<vector<int>> a(n, vector<int>(m, 5));

// 每行大小可以不同（锯齿数组）
vector<vector<int>> a(n);
a[0] = vector<int>(3);  // 第一行3列
a[1] = vector<int>(5);  // 第二行5列
```
