# 数组方法的语法解释

代码：

```c++
bool seen[401] = {false};  // 这行是什么意思？
int distinct = 0;

for (int i = top; i <= bottom; i++) {
    for (int j = left; j <= right; j++) {
        int color = a[i][j];
        if (!seen[color]) {      // 这行是什么意思？
            seen[color] = true;  // 这行是什么意思？
            distinct++;
        }
    }
}
```

逐行解释：

1. `bool seen[401] = {false};`

   这行代码的作用：

   ```c++
   bool seen[401];      // ① 创建一个能存储401个布尔值的数组
   seen[401] = {false}; // ② 把所有元素初始化为 false
   ```

   **具体含义：**

   - `bool` 是布尔类型，只有两个值：`true`（真）和 `false`（假）
   - `seen[401]` 创建了一个名为 `seen` 的数组，有 401 个位置
   - `= {false}` 把数组的**所有元素**都初始化为 `false`

   **索引范围：** `seen[0]` 到 `seen[400]`，共 401 个位置

   **为什么是 401？**

   - 因为题目说颜色值 `a[i][j] ≤ n×m`，而 `n,m ≤ 20`，所以最大颜色值 ≤ 400
   - 为了能用 `seen[color]` 访问，数组大小至少要是 `color最大值 + 1`
   - 如果颜色值是 400，就需要 `seen[400]`，所以数组大小是 401

2. `if (!seen[color])`

   这是条件判断语句，分成两部分理解：

   ```c++
   seen[color]  // 访问数组中索引为 color 的元素，得到一个布尔值
   !seen[color] // "!" 是取反操作符，把 true 变 false，false 变 true
   ```

   **举例说明：**

   ```c++
   bool seen[401] = {false};  // 所有元素都是 false
   
   int color = 5;
   cout << seen[5];   // 输出 0 (false)
   cout << !seen[5];  // 输出 1 (true)
   
   if (!seen[5]) {    // 因为 seen[5] 是 false，!false = true，条件成立
       cout << "颜色5还没出现过";
   }
   ```

3. `seen[color] = true;`

   把数组中索引为 `color` 的位置设为 `true`，标记这个颜色已经出现过了。

---

执行过程示例：

假设子矩阵中的颜色为：

```
1 2 3
2 3 4
```

**执行过程：**

```c++
// 初始状态
bool seen[401] = {false};  // 所有位置都是 false
int distinct = 0;

// 第1次循环：color = 1
if (!seen[1]) {  // seen[1] 是 false，!false = true，条件成立
    seen[1] = true;  // 标记颜色1已出现
    distinct++;      // distinct = 1
}

// 第2次循环：color = 2
if (!seen[2]) {  // seen[2] 是 false，条件成立
    seen[2] = true;  // 标记颜色2已出现
    distinct++;      // distinct = 2
}

// 第3次循环：color = 3
if (!seen[3]) {  // seen[3] 是 false，条件成立
    seen[3] = true;  // 标记颜色3已出现
    distinct++;      // distinct = 3
}

// 第4次循环：color = 2
if (!seen[2]) {  // seen[2] 现在是 true，!true = false，条件不成立
    // 不执行里面的代码
}

// 第5次循环：color = 3
if (!seen[3]) {  // seen[3] 是 true，条件不成立
    // 不执行里面的代码
}

// 第6次循环：color = 4
if (!seen[4]) {  // seen[4] 是 false，条件成立
    seen[4] = true;  // 标记颜色4已出现
    distinct++;      // distinct = 4
}

// 最终 distinct = 4，表示有4种不同颜色
```
