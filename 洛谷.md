## B4548 成绩补录*

### 题目描述

学校一共有 *n* 门课程，每门课程都有一个课程代码。

接下来会录入 *m* 条成绩记录。每条记录包含一个课程代码 *c* 和一个成绩 *s*。如果同一门课程被录入多次，系统只保留最后一次录入的成绩。

录入结束后，你需要输出没有录入成绩的课程数量，以及已录入成绩但成绩低于 60 分的课程数量。

### 输入格式

第一行两个整数 *n*,*m*。

第二行有 *n* 个整数，依次表示每门课程的课程代码。

接下来 *m* 行，每行两个整数 *c*,*s*，表示课程代码为 *c* 的课程录入了成绩 *s*。

### 输出格式

一行输出两个整数，分别表示没有录入成绩的课程数量和已录入成绩但成绩低于 60 分的课程数量。

### 输入输出样例

**输入 #1**复制运行

```
4 5
101 102 103 104
101 80
102 50
101 90
104 59
102 70
```

**输出 #1**复制运行

```
1 1
```

**输入 #2**复制运行

```
3 0
11 22 33
```

**输出 #2**复制运行

```
3 0
```

**输入 #3**复制运行

```
5 4
1 2 3 4 5
5 100
3 40
4 60
3 55
```

**输出 #3**复制运行

```
2 1
```

### 说明/提示

样例 1 中，课程 101 最后成绩为 90，课程 102 最后成绩为 70，课程 103 没有成绩，课程 104 成绩为 59。因此没有录入成绩的课程有 1 门，低于 60 分的课程有 1 门。

样例 3 中，课程 1,2 没有成绩；课程 3 最后成绩为 55，这个成绩低于 60。

- 对于 30% 的数据，保证：所有录入的成绩均 ≥60。
- 对于另外 30% 的数据，保证：每门课程最多被录入一次。
- 对于 100% 的数据，保证：1≤*n*≤1000，0≤*m*≤1000，课程代码均为不超过 100000 的正整数，0≤*s*≤100。保证所有课程的课程代码互不相同。保证录入成绩时课程代码存在。

### 思路

1. 记录所有课程
2. 处理m条成绩记录，同一门课程只保留最后一条
3. 统计两个数据：
   - 没有录入的课程数量
   - 已经录入成绩但<60的课程数量

### 解题

#### 法1：数组（数据范围小）

由于n<=100000,可以用数组直接储存

```c++
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    
    vector<int> courses(n);  // 存储所有课程代码
    vector<int> scores(n, -1);  // 存储每个课程的成绩，-1表示未录入
    
    // 读入课程代码
    for (int i = 0; i < n; i++) {
        cin >> courses[i];
    }
    
    // 处理成绩录入
    for (int i = 0; i < m; i++) {
        int c, s;
        cin >> c >> s;
        
        // 找到该课程在数组中的位置
        for (int j = 0; j < n; j++) {
            if (courses[j] == c) {
                scores[j] = s;  // 更新成绩（覆盖之前的）
                break;
            }
        }
    }
    
    // 统计结果
    int noScore = 0;      // 没有成绩的课程数
    int below60 = 0;      // 低于60分的课程数
    
    for (int i = 0; i < n; i++) {
        if (scores[i] == -1) {
            noScore++;
        } else if (scores[i] < 60) {
            below60++;
        }
    }
    
    cout << noScore << " " << below60 << endl;
    
    return 0;
}
```

## B3690 [语言月赛202212] 盒武器*

### 题目描述

Aya 在网校共有两名助教，她们的网名分别为某 E 和 L 队。

Aya 热衷于开盒。他对两名助教使用了盒武器，得知某 E 的真实姓名为一个字符串 *s*，L 队的真实姓名为一个字符串 *t*。两个人的名字均只包含英文小写字母。

Aya 确信实力和姓名的字典序有关，姓名字典序越靠前，实力越强。

但是事实上，L 队的实力严格强于某 E。所以，你需要重新确定 26 个英文小写字母的大小顺序，以保证 Aya 得到的结果与事实相符。

也就是说，请你重新定义 26 个字母之间的大小关系，使得按照你新定义的大小关系比较 *s*,*t* 两字符串的字典序时，满足 *t*<*s*。

字符串 *t* 的字典序小于 *s* 的字典序当且仅当满足如下两个条件之一：

1. *t* 是 *s* 的一个前缀；
2. 存在一个位置 *j*≤min(∣*s*∣,∣*t*∣)，使得对 1≤*i*<*j* 都有 *s**i*=*t**i* 且 *t**j*<*s**j*。其中两字符相比较的小于号是你重新定义的小于关系。

### 输入格式

输入的第一行为一个字符串 *s*。
输入的第二行为一个字符串 *t*。

### 输出格式

输出一行一个字符串，长度为 26，a∼z 这些字符在你的输出中均应恰好出现一次，以表示字母之间新定义的大小关系。

在输出中靠前位置的字母小于在输出中靠后位置的字母。

答案可能有多种，你只需要给出任意一组。

### 输入输出样例

**输入 #1**复制运行

```
b
a
```

**输出 #1**复制运行

```

### 思路

#### 题目分析

这道题要求重新定义26个字母的大小顺序，使得在自定义情况下，字符串t字典序小于t

#### 核心思路

##### 字典序比较规则

当比较两个字符串s,t时，字典序t<s需要满足：

- t是s前缀：例如t='ab',s='abcd'
- 在第一个不同的位置j，t[j]<s[j]

题目保证解存在，所以需要：

- 找到s,t的第一个不同字符
- 让t的这个字符小于s的字符

其他字符的顺序可以随意排列

#### 解题步骤

1. 找到第一个不同的位置

   ```c++
   int position = 0;
   while (position < s.length() && position < t.length() && s[position] == t[position]){
   position++;
   }
   ```

2. 确定字符大小关系

   1. 在某一位置字符不同
   2. 一个字符串是另一个的前缀（此时t<s自动成立，这种情况下任何字母顺序都可以

3. 构造新字母顺序

   输出长度为26的排序，保证：

   - t[position]在s[position]前面
   - 其他字母任意排列

#### 解题

#### 法1：简单实现

```c++
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string s, t;
    cin >> s >> t;
    
    // 找到第一个不同的位置
    int pos = 0;
    while (pos < s.length() && pos < t.length() && s[pos] == t[pos]) {
        pos++;
    }
    
    // 如果 pos 到达某个字符串末尾，说明一个是另一个的前缀
    // 此时任意顺序都可以
    string order = "abcdefghijklmnopqrstuvwxyz";
    
    // 如果找到了不同字符，需要让 t[pos] 在 s[pos] 前面
    if (pos < s.length() && pos < t.length()) {
        char smaller = t[pos];
        char larger = s[pos];
        
        // 将 order 中 larger 移到 smaller 后面
        // 方法：把 larger 删除，插入到 smaller 后面
        string newOrder = "";
        for (char c : order) {
            if (c == larger) continue;  // 跳过 larger
            newOrder += c;
            if (c == smaller) {
                newOrder += larger;  // 在 smaller 后面插入 larger
            }
        }
        order = newOrder;
    }
    
    cout << order << endl;
    return 0;
}
```

#### 法2：更简单实现

```c++
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s, t;
    cin >> s >> t;
    
    // 基础顺序（可以是任何顺序）
    string order = "abcdefghijklmnopqrstuvwxyz";
    
    // 找到第一个不同的位置
    int pos = 0;
    while (pos < s.size() && pos < t.size() && s[pos] == t[pos]) {
        pos++;
    }
    
    // 如果两个字符串在某个位置不同
    if (pos < s.size() && pos < t.size()) {
        char a = t[pos];  // 需要小的字符
        char b = s[pos];  // 需要大的字符
        
        // 确保 a 在 b 前面
        int idxA = order.find(a);
        int idxB = order.find(b);
        
        if (idxA > idxB) {
            // 如果 a 在 b 后面，交换它们的位置
            swap(order[idxA], order[idxB]);
        }
    }
    // 如果是前缀关系，不需要修改顺序
    
    cout << order << endl;
    return 0;
}
```

### 问题分析

#### 1. n未初始化就使用，会RE

```c++
int n,x,y;
vector<int> a(n);  // ❌ n 还没有值！可能崩溃
cin >> n >> x >> y;
```

#### 2. len大小错误

```c++
vector<int> len(n);  // ❌ 和 a 一样大，但满足条件的数可能更少
```

如果满足条件的数少于 n，`len` 中会有多余的 0。

应该：使用动态添加

```c++
vector<int> len;  // 先不指定大小
// 满足条件时：
len.push_back(z);
```

#### 3. num变量逻辑错误

```c++
int num = 0;  // 每次循环都重置为 0
if (条件) {
    num++;  // num 变成 1
    len[i] = z;
}
if (num > 0) {
    cout << num << ...;  // num 永远是 1
}
```

**问题**：

- `num` 每次循环重置，无法统计总数
- 每个满足条件的数都会输出一次，导致输出多行

**修正**：用 `count` 变量统计总数

```c++
int count = 0;  // 放在循环外面
if (条件) {
    count++;
    len.push_back(z);
}
// 最后统一输出
```

#### 4. max_element使用错误

```c++
*max_element(len.begin(), len.end())  // ❌ 如果 len 为空会崩溃
```

当没有满足条件的数时，`len` 为空，解引用空迭代器会崩溃。

**修正**：先判断是否为空

```c++
if (!len.empty()) {
    int maxLen = *max_element(len.begin(), len.end());
}
```

#### 5. 输出在循环内部

```c++
for (int i = 0; i < n; i++) {
    // ...
    if (num > 0) {
        cout << num << ...;  // ❌ 每找到一个就输出一次
    }
}
```

**问题**：会输出多行，不符合题目要求（只输出一行）。

#### 6. 输出没有空格分开

```c++
cout << num << *max_element(len.begin(), len.end()) << endl;
//      ↑没有空格分隔
```

**输出示例**：`18`（你以为是 "1 8"，实际是 "18"）

### 思路

#### 题目分析

略

#### 核心逻辑

对每个z=a[i]检查6个条件

#### 代码实现

##### 法1：暴力判断

```c++
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n, x, y;
    cin >> n >> x >> y;
    
    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    
    int count = 0;      // 满足条件的树枝数量
    int maxLen = 0;     // 满足条件的最长树枝
    
    for (int i = 0; i < n; i++) {
        int z = a[i];
        
        // 检查是否满足条件
        bool isValid = false;
        
        // 检查：某两根之和等于第三根
        if (x + y == z || x + z == y || y + z == x) {
            isValid = true;
        }
        // 检查：某两根之积等于第三根
        else if (x * y == z || x * z == y || y * z == x) {
            isValid = true;
        }
        
        if (isValid) {
            count++;
            if (z > maxLen) {
                maxLen = z;
            }
        }
    }
    
    cout << count << " " << maxLen << endl;
    return 0;
}

### 问题

#### **`a.find(0)` 错误**

```c++
k = a.find(0);  // ❌ 错误
```

- `vector` 没有 `find` 成员函数
- 应该使用 `std::find` 算法，或者直接遍历

#### **输出运算符错误**

```c++
cout >> k >> endl;  // ❌ 错误
```

- 应该用 `<<` 而不是 `>>`
- `>>` 是输入运算符，`<<` 是输出运算符
- 正确写法：`cout << k << endl;`

#### **`find(0)` 返回的是迭代器，不是索引**

即使使用正确的 `std::find`：

```c++
auto it = find(a.begin(), a.end(), 0);
k = it;  // ❌ 错误：迭代器不能直接赋值给int
```

应该计算距离：

```c++
k = it - a.begin();  // 得到索引
```

#### **find 会找到自己（如果 a[k]==0）**

你的逻辑是"如果 a[k] 被占用，就找第一个0"，但实际上 `a[k]!=0` 时才会找，逻辑是对的，但实现不对。（对的对的，既然如此，我的思路可行）

#### 修正后代码（使用std::find)

```c++
#include <iostream>
#include <vector>
#include <algorithm>  // 需要包含这个
using namespace std;

int main(){
    int n, k;
    cin >> n;
    vector<int> a(n + 1);
    
    for(int i = 1; i <= n; i++){
        cin >> a[i];
    }
    
    cin >> k;
    
    if (a[k] != 0){
        // 从索引1开始找第一个0
        auto it = find(a.begin() + 1, a.end(), 0);
        if (it != a.end()){
            k = it - a.begin();  // 迭代器转索引
        }
    }
    
    cout << k << endl;  // 注意是 << 不是 >>
    return 0;
}

### 错误

#### 1. vector初始化错误

```c++
vector<string> yuan("a","e","i","o","u");  // ❌ 错误
```

应使用花括号或者push_back:

```c++
vector<string> yuan = {"a","e","i","o","u"};  // ✅ 正确
// 或者
vector<char> yuan = {'a','e','i','o','u'};   // 更好，因为字符比较
```

#### 2. find方法使用错误

vector没有find方法，应该使用std::find:

```c++
if (find(yuan.begin(), yuan.end(), s[j]) == yuan.end())
```

- 注：[关于.find()和std的疑惑解答]("D:/VScode/c++/关于std的拓展.md")

- [关于.find()和std的疑惑解答]("./关于std的拓展.md")

- [关于.find()和std的疑惑解答](../../VScode/c++/关于std的拓展.md)

- 在c++中，有自己的find成员方法的容器有：

  1. 关联容器（基于树或哈希表）

     ```c++
     std::set<int> s = {1, 2, 3, 4};
     auto it = s.find(3);  // ✅ O(log n)
     
     std::map<string, int> m = {{"a", 1}, {"b", 2}};
     auto it = m.find("a");  // ✅ O(log n)
     
     std::unordered_set<int> us = {1, 2, 3};
     auto it = us.find(2);  // ✅ O(1) 平均
     
     std::unordered_map<string, int> um = {{"a", 1}};
     auto it = um.find("a");  // ✅ O(1) 平均
     ```

  2. 字符串

     ```c++
     std::string str = "hello world";
     size_t pos = str.find("world");  // ✅ 返回位置索引
     size_t pos2 = str.find('o');     // ✅ 查找字符
     ```

  没有find（）成员方法的容器（需要使用std::find):

  1. 序列容器

     ```c++
     std::vector<int> v = {1, 2, 3, 4};
     auto it = std::find(v.begin(), v.end(), 3);  // ✅ O(n)
     
     std::list<int> l = {1, 2, 3};
     auto it = std::find(l.begin(), l.end(), 2);  // ✅ O(n)
     
     std::deque<int> d = {1, 2, 3};
     auto it = std::find(d.begin(), d.end(), 2);  // ✅ O(n)
     
     std::array<int, 3> a = {1, 2, 3};
     auto it = std::find(a.begin(), a.end(), 2);  // ✅ O(n)
     ```

  为什么出现这样的区别：

  | 容器类型                        | 数据结构 | 查找复杂度 | 有find成员？     |
  | :------------------------------ | :------- | :--------- | :--------------- |
  | `vector`                        | 动态数组 | O(n)       | ❌ 用 `std::find` |
  | `list`                          | 双向链表 | O(n)       | ❌ 用 `std::find` |
  | `deque`                         | 双端队列 | O(n)       | ❌ 用 `std::find` |
  | `set`/`map`                     | 红黑树   | O(log n)   | ✅ 有成员         |
  | `unordered_set`/`unordered_map` | 哈希表   | O(1) 平均  | ✅ 有成员         |
  | `string`                        | 字符串   | O(n)       | ✅ 有成员（特殊） |

  设计原因：

  - 关联容器利用自身数据结构特性，可以实现更高效的查找，所以提供成员方法
  - 序列容器只能线性查找，用std::find通用算法足够，不需要重复实现

  使用建议：

  ```c++
  // 对于 vector/list/deque 等
  std::vector<int> v = {1, 2, 3};
  if (std::find(v.begin(), v.end(), 2) != v.end()) {
      // 找到了
  }
  
  // 对于 set/map 等
  std::set<int> s = {1, 2, 3};
  if (s.find(2) != s.end()) {
      // 找到了
  }
  ```

- 注：std是standard的缩写，在c++中代表标准命名空间

  - 简单理解：std类似于一个大仓库存放了c++标准库提供的所有功能如（输入输出cout,cin,容器vector,string,算法find,sort...)

  - 为什么用：由于c++中有太多名字，为了避免命名冲突，就都放在std中，比如：

    ```c++
    // 假设你自己也定义了一个叫 find 的函数
    int find(int x) { return x * 2; }
    
    // 但你想用标准库的 find
    std::find(v.begin(), v.end(), 3);  // 用 std:: 明确表示：我要用标准库的 find
    ```

  - 使用方法：

    1. 每次加std::

       ```c++
       #include <iostream>
       int main() {
           std::cout << "Hello" << std::endl;  // 明确告诉编译器用的是标准库的 cout
           return 0;
       }
       ```

    2. 使用using namespace std;

       ```c++
       #include <iostream>
       using namespace std;  // 把 std 仓库全部打开
       int main() {
           cout << "Hello" << endl;  // 不用写 std:: 了
           return 0;
       }
       ```

       **缺点**：如果自己定义了和标准库同名的函数，会产生冲突。

    3. 只引入需要的

       ```c++
       #include <iostream>
       using std::cout;
       using std::endl;
       int main() {
           cout << "Hello" << endl;  // 只对 cout 和 endl 免前缀
           std::cin >> x;            // cin 还是要加 std::
           return 0;
       }
       ```

#### 4. 字符串方法错误

```c++
cout << s[j+1].upper()  // ❌ 错误
```

- upper() 不是c++字符串的方法
- s[j+1]是char类型，没有upper()方法

要转为大写应使用 toupper()

```c++
cout << (char)toupper(s[j+1]);
```

#### 5. 多余的引号

```c++
cout << s[j+1].upper()<< "";  // << "" 是多余的
cout << "" << endl;           // 直接 cout << endl;
```

### 修正代码

```c++
#include <iostream>
#include <vector>
#include <algorithm>
#include <string>
#include <cctype>  // 需要 toupper
using namespace std;

int main(){
    int n;
    cin >> n;
    for (int i = 0; i < n; i++){
        string s;
        cin >> s;
        vector<char> yuan = {'a','e','i','o','u'};
        
        for (int j = 0; j < (int)s.length() - 1; j++){
            // 检查当前字符是否为元音
            if (find(yuan.begin(), yuan.end(), s[j]) == yuan.end()){
                continue;
            } else {
                // 输出下一个字母的大写形式
                cout << (char)toupper(s[j + 1]);
            }
        }
        cout << endl;
    }
    return 0;
}
```

注意事项：

- 如果 `s.length() == 0`，`s.length() - 1` 会溢出，建议加保护
- 确保输入字符串不为空
