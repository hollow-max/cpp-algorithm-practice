# 洛谷

## B4548 成绩补录

### 题目描述

学校一共有 *n* 门课程，每门课程都有一个课程代码。

接下来会录入 *m* 条成绩记录。每条记录包含一个课程代码 *c* 和一个成绩 *s*。如果同一门课程被录入多次，系统只保留最后一次录入的成绩。

录入结束后，你需要输出没有录入成绩的课程数量，以及已录入成绩但成绩低于 60 分的课程数量。

### 输入格式

第一行两个整数 *n*,*m*。

第二行有 *n* 个整数，依次表示每门课程的课程代码。

接下来 *m* 行，每行两个整数 *c*,*s*，表示课程代码为 *c* 的课程录入了成绩 *s*。

### 输出格式

一行输出两个整数，分别表示没有录入成绩的课程数量和已录入成绩但成绩低于 60 分的课程数量。

### 输入输出样例

**输入 #1**复制运行

```
4 5
101 102 103 104
101 80
102 50
101 90
104 59
102 70
```

**输出 #1**复制运行

```
1 1
```

**输入 #2**复制运行

```
3 0
11 22 33
```

**输出 #2**复制运行

```
3 0
```

**输入 #3**复制运行

```
5 4
1 2 3 4 5
5 100
3 40
4 60
3 55
```

**输出 #3**复制运行

```
2 1
```

### 说明/提示

样例 1 中，课程 101 最后成绩为 90，课程 102 最后成绩为 70，课程 103 没有成绩，课程 104 成绩为 59。因此没有录入成绩的课程有 1 门，低于 60 分的课程有 1 门。

样例 3 中，课程 1,2 没有成绩；课程 3 最后成绩为 55，这个成绩低于 60。

- 对于 30% 的数据，保证：所有录入的成绩均 ≥60。
- 对于另外 30% 的数据，保证：每门课程最多被录入一次。
- 对于 100% 的数据，保证：1≤*n*≤1000，0≤*m*≤1000，课程代码均为不超过 100000 的正整数，0≤*s*≤100。保证所有课程的课程代码互不相同。保证录入成绩时课程代码存在。

### 思路

1. 记录所有课程
2. 处理m条成绩记录，同一门课程只保留最后一条
3. 统计两个数据：
   - 没有录入的课程数量
   - 已经录入成绩但<60的课程数量

### 解题

#### 法1：数组（数据范围小）

由于n<=100000,可以用数组直接储存

```c++
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    
    vector<int> courses(n);  // 存储所有课程代码
    vector<int> scores(n, -1);  // 存储每个课程的成绩，-1表示未录入
    
    // 读入课程代码
    for (int i = 0; i < n; i++) {
        cin >> courses[i];
    }
    
    // 处理成绩录入
    for (int i = 0; i < m; i++) {
        int c, s;
        cin >> c >> s;
        
        // 找到该课程在数组中的位置
        for (int j = 0; j < n; j++) {
            if (courses[j] == c) {
                scores[j] = s;  // 更新成绩（覆盖之前的）
                break;
            }
        }
    }
    
    // 统计结果
    int noScore = 0;      // 没有成绩的课程数
    int below60 = 0;      // 低于60分的课程数
    
    for (int i = 0; i < n; i++) {
        if (scores[i] == -1) {
            noScore++;
        } else if (scores[i] < 60) {
            below60++;
        }
    }
    
    cout << noScore << " " << below60 << endl;
    
    return 0;
}
```

#### 法2：`unordered_map`(更高效)

数据范围更大时，使用哈希表更好

```c++
#include <iostream>
#include <unordered_map>
#include <vector>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    
    vector<int> courses(n);
    unordered_map<int, int> scoreMap;  // 课程代码 -> 成绩
    
    // 读入课程代码
    for (int i = 0; i < n; i++) {
        cin >> courses[i];
    }
    
    // 处理成绩录入
    for (int i = 0; i < m; i++) {
        int c, s;
        cin >> c >> s;
        scoreMap[c] = s;  // 自动覆盖之前的成绩
    }
    
    // 统计结果
    int noScore = 0;
    int below60 = 0;
    
    for (int code : courses) {
        if (scoreMap.find(code) == scoreMap.end()) {
            noScore++;  // 没有成绩
        } else if (scoreMap[code] < 60) {
            below60++;  // 成绩低于60
        }
    }
    
    cout << noScore << " " << below60 << endl;
    
    return 0;
}
```

1. `unordered_map<int, int> scoreMap;`c++标准库中的一个哈希表容器，用于储存键值对

   - ```c++
     unordered_map<键的类型, 值的类型> 变量名;
     unordered_map<int, int> scoreMap;  
     // 键的类型：int（课程代码）
     // 值的类型：int（成绩）
     // 变量名：scoreMap
     ```

   - ```c++
     #include <iostream>
     #include <unordered_map>
     using namespace std;
     
     int main() {
         // 创建哈希表
         unordered_map<int, int> scoreMap;
         
         // 插入数据
         scoreMap[101] = 80;
         scoreMap[102] = 50;
         scoreMap[101] = 90;  // 覆盖前面的80
         
         cout << "课程101的成绩：" << scoreMap[101] << endl;  // 90
         cout << "课程102的成绩：" << scoreMap[102] << endl;  // 50
         
         // 检查键是否存在
         if (scoreMap.find(103) == scoreMap.end()) {
             cout << "课程103没有成绩" << endl;
         }
         
         // 遍历所有数据
         cout << "\n所有成绩：" << endl;
         for (auto& pair : scoreMap) {
             cout << "课程 " << pair.first << " => " << pair.second << " 分" << endl;
         }
         
         // 获取大小
         cout << "\n共 " << scoreMap.size() << " 门课程有成绩" << endl;
         
         return 0;
     }
     ```

   - ```
     课程101的成绩：90
     课程102的成绩：50
     课程103没有成绩
     
     所有成绩：
     课程 102 => 50 分
     课程 101 => 90 分
     
     共 2 门课程有成绩
     ```

2. 注意：使用unordered_map要#include

3. 注意：注意变量不要打错

#### 法3：`map`(自动排序)

```c++
#include <iostream>
#include <map>
#include <vector>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    
    vector<int> courses(n);
    map<int, int> scoreMap;  // 课程代码 -> 成绩（自动排序）
    
    for (int i = 0; i < n; i++) {
        cin >> courses[i];
    }
    
    for (int i = 0; i < m; i++) {
        int c, s;
        cin >> c >> s;
        scoreMap[c] = s;
    }
    
    int noScore = 0;
    int below60 = 0;
    
    for (int code : courses) {
        if (scoreMap.count(code) == 0) {
            noScore++;
        } else if (scoreMap[code] < 60) {
            below60++;
        }
    }
    
    cout << noScore << " " << below60 << endl;
    
    return 0;
}
```
