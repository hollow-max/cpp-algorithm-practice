十天不够“系统学完 C++ 和算法”，但够你把集训里最容易卡死的基础补到能跟上。目标要改成：**不追求全会，先保证听课能懂、代码能写、简单题能独立 AC。**

我建议你这十天按“救急路线”走：

**第 1-2 天：C++ 生存语法**
重点只学这些：输入输出、`if/for/while`、数组、函数、`vector`、`string`、`sort`。

你要能熟练写：

```c++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    vector<int> a(n);

    for (int i = 0; i < n; i++) cin >> a[i];

    sort(a.begin(), a.end());

    for (int x : a) cout << x << ' ';
    return 0;
}
```

**第 3-4 天：模拟、枚举、排序**
这是集训最基础也最常见的部分。
你要练会：按题意一步步写、遍历所有可能、排序后处理。

关键词：`for`、数组下标、最大值最小值、计数、排序。

**第 5 天：前缀和**
必须会，性价比极高。

```c++
vector<long long> pre(n + 1);
for (int i = 1; i <= n; i++) {
    cin >> a[i];
    pre[i] = pre[i - 1] + a[i];
}

// 区间 [l, r] 的和
cout << pre[r] - pre[l - 1] << '\n';
```

**第 6 天：双指针**
会基本滑动窗口即可。
先记住一句话：右指针扩展，左指针收缩。

**第 7 天：二分**
先会二分答案和查找位置。重点不是背代码，而是理解：
“答案具有单调性，就可能二分。”

**第 8 天：DFS / BFS**
集训很可能讲搜索。
你至少要会：递归 DFS、队列 BFS、网格四方向。

**第 9 天：动态规划入门**
不要一上来啃难 DP。只学：

- 斐波那契型
- 最大子段和
- 01 背包的思想

**第 10 天：总复习 + 整理模板**
不要学新东西。
只做三件事：

1. 把常用模板手敲一遍。
2. 复盘错题。
3. 准备一个自己的 `contest_template.cpp`。

你现在最该避免的是：今天看图论，明天看线段树，后天看数论。会非常焦虑，而且记不住。你现在要做的是把低层能力补起来：**读入、循环、数组、STL、基础模板。**

每天建议这样安排：

```txt
30 分钟：看语法/模板
90 分钟：手敲例题
120 分钟：做 3-5 道简单题
30 分钟：整理错因和模板
```

如果时间更紧，每天至少保证：**手敲代码 2 小时**。只看视频几乎没用，算法比赛是肌肉记忆活儿。
